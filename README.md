# SQL Data Analysis Practice

## Overview
This repository contains structured SQL queries written using PostgreSQL to practice data analysis concepts and business problem solving.

## Concepts Covered
- Basic SELECT statements
- Filtering using WHERE
- Sorting using ORDER BY
- Aggregate functions (SUM, AVG, COUNT, MAX, MIN)
- GROUP BY and HAVING
- Joins (INNER JOIN, LEFT JOIN)
- Subqueries
- CASE statements
- Window functions

## Purpose
The objective of this repository is to strengthen SQL fundamentals for data analysis and business intelligence roles.
